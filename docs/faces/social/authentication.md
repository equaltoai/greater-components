# Social Face: Authentication

Social face components commonly pair with the shared auth module for login, signup, and session handling.

## Shared Auth Module

```ts
import * as Auth from '@equaltoai/greater-components/shared/auth';
```

## CLI Install

```bash
greater add shared/auth
```

## Notes

- When using `faces/social`, import its stylesheet after primitives: `@equaltoai/greater-components/faces/social/style.css`.
