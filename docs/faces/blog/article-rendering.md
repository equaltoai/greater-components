# Blog Face: Article Rendering

The blog face uses the `content` package for markdown rendering (e.g. `MarkdownRenderer`).

## Article compound component

Import from:

```ts
import { Article } from '@equaltoai/greater-components/faces/blog';
```

Common subcomponents include:

- `Article.Header`
- `Article.Content`
- `Article.TableOfContents`
- `Article.RelatedPosts`
