# @equaltoai/greater-components-icons

Icon components for Greater Components.

## Usage (recommended)

```svelte
<script>
	import { SearchIcon, SettingsIcon } from '@equaltoai/greater-components/icons';
</script>

<SearchIcon />
<SettingsIcon />
```

## Standalone package

```ts
import { SearchIcon } from '@equaltoai/greater-components-icons';
```
