# @equaltoai/greater-components-compose

Post/content composer components for Greater Components.

## Usage (recommended)

```svelte
<script>
	import { Compose } from '@equaltoai/greater-components/shared/compose';
</script>

<Compose.Root>
	<Compose.Editor />
	<Compose.CharacterCount />
	<Compose.Submit />
</Compose.Root>
```

## Standalone package

```ts
import { Compose } from '@equaltoai/greater-components-compose';
```
