# @equaltoai/greater-components-auth

Authentication components for Greater Components.

## Usage (recommended)

```svelte
<script>
	import * as Auth from '@equaltoai/greater-components/shared/auth';
</script>

<Auth.Root>
	<Auth.LoginForm />
</Auth.Root>
```

## Standalone package

```svelte
<script>
	import * as Auth from '@equaltoai/greater-components-auth';
</script>
```
