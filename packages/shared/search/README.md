# @equaltoai/greater-components-search

Search components for Greater Components.

## Usage (recommended)

```svelte
<script>
	import * as Search from '@equaltoai/greater-components/shared/search';
</script>

<Search.Root>
	<Search.Bar />
	<Search.Filters />
	<Search.Results />
</Search.Root>
```

## Standalone package

```ts
import * as Search from '@equaltoai/greater-components-search';
```
