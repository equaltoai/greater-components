### Module: Create Chat Demo Page
Create the main chat demo page with full chat interface demonstration.

## Description
Create a comprehensive chat demo page at `apps/playground/src/routes/chat/+page.svelte` that demonstrates all chat components from the `@equaltoai/greater-components-chat` package. The demo includes:

1. **Full Chat Interface**: Chat.Container wrapping Chat.Header, Chat.Messages, Chat.Suggestions, and Chat.Input
2. **Mock Message Data**: Sample conversation with user and assistant messages including tool calls
3. **Simulated Streaming**: setTimeout-based typing effect with cursor animation
4. **Tool Call Demonstration**: Chat.ToolCall components showing pending, running, and complete states
5. **Settings Modal**: Button to open Chat.Settings with model selection and configuration
6. **Theme Support**: Works with light/dark mode via ThemeProvider
7. **Code Examples**: Usage snippets showing import statements and basic/advanced patterns

CRITICAL CORRECTIONS from original spec:
- Import path is `@equaltoai/greater-components-chat` (NOT `@equaltoai/greater-components/chat`)
- Types are imported from the chat package directly
- ChatMessages uses context for messages, not direct props
- ChatToolCall error demo had syntax error (missing `error:` key)
- Removed invalid `renderMarkdown` and `loading` props from ChatMessages

	async function simulateStreaming(content: string): Promise<void> {
		isStreaming = true;

			streamContent += chars[i];
			
			// Update the message content

					: msg
