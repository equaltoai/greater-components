# @equaltoai/greater-components-chat

## 4.0.0

### Major Changes

- Initial release of the Chat component suite for Greater Components

  ## Features

  ### Components
  - **Container** - Root container component with context provider for chat state management
  - **Message** - Individual message display with support for user, assistant, and system roles
  - **Messages** - Messages list container with auto-scroll functionality
  - **Input** - Text input component for composing messages with submit handling
  - **ToolCall** - Display component for AI tool calls with status indicators
  - **Suggestions** - Quick suggestion buttons for common prompts
  - **Header** - Header component with title and optional actions
  - **Settings** - Settings modal component for configuring chat behavior

  ### Context System
  - `createChatContext()` - Factory function for creating chat context
  - `getChatContext()` - Retrieve current chat context
  - `setChatContext()` - Set chat context value
  - `hasChatContext()` - Check if chat context exists

  ### Utilities
  - `formatMessageTime()` - Format message timestamps
  - `formatStreamDuration()` - Format streaming response duration
  - `getConnectionStatusText()` - Get human-readable connection status

  ### TypeScript Support
  - Full type definitions for all components and utilities
  - Exported types: `MessageRole`, `MessageStatus`, `ToolCallStatus`, `ToolCall`, `ChatMessage`, `ChatContainerProps`, `ChatMessageProps`, `ChatMessagesProps`, `ChatInputProps`, `ChatToolCallProps`, `ChatSuggestion`, `ChatSuggestionItem`, `ChatSuggestionsProps`, `ChatHeaderProps`, `ChatSettingsState`, `ChatSettingsProps`, `KnowledgeBaseConfig`, `ChatHandlers`, `ChatState`, `ChatContextValue`, `ConnectionStatus`

  ## Usage

  ```svelte
  <script>
    import * as Chat from '@equaltoai/greater-components/chat';

    const handlers = {
      onSubmit: async (content) => {
        const response = await fetch('/api/chat', {
          method: 'POST',
          body: JSON.stringify({ message: content }),
        });
        return response.json();
      },
    };
  </script>

  <Chat.Container {handlers}>
    <Chat.Header title="AI Assistant" />
    <Chat.Messages />
    <Chat.Input placeholder="Type a message..." />
  </Chat.Container>
  ```

  ## Dependencies
  - `@equaltoai/greater-components-primitives` - Core UI components
  - `@equaltoai/greater-components-icons` - Icon components
  - `@equaltoai/greater-components-headless` - Headless component behaviors
  - `@equaltoai/greater-components-content` - Content rendering (CodeBlock, MarkdownRenderer)
>>> | TYPE: content creation | DESC: Create initial CHANGELOG.md for chat package documenting v4.0.0 release
