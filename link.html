External link (target=_blank)

When target="_blank" is set without a consumer-supplied rel, Link auto-injects rel="noopener noreferrer" to prevent reverse tabnabbing.

Inline link inside body text

Use variant="inline" for links that appear in body text. The inline variant inherits font size and family from its surroundings and underlines by default.

See the Usage page for current consumption, or external documentation for deeper reference. Inline links work mid-sentence without breaking the line flow.

Active route (aria-current="page")

Consumers wire aria-current="page" from their router's active-route state. Link applies a [aria-current="page"] CSS hook for distinct visual treatment (bolder font-weight by default).

Deliberate non-features

Two things Link intentionally does NOT do:

  • No Space-key activation. Space on <a> is intentionally not native (Space scrolls the page). Intercepting it would be an a11y anti-pattern that surprises users mid-typing.
  • No disabled prop. Disabled navigation is an a11y anti-pattern; consumers should remove the link or use a different affordance when navigation is unavailable.
